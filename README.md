# toc_project
Django website for regular expression and context free grammar.

## how to install :

Install the python packages 
1. Django 
2. pyformlang

pip install package_name

use the command 

* python manage.py make migrations
* python manage.py runserver 

## Input Format :
In regular expression : input should be given the form of concatenation enclosed in simple brackets 
union enclosed in square brackets.

In context free grammar : In place of epsilon , the given space should be left blank.
